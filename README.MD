# AD300 WK 1 Assignments
This repo contains a class `GradesToolBox` to handle GPA to % conversion.

## Build and Run
Build using maven
`mvn compile`

Run using maven
` mvn exec:java -Dexec.mainClass="org.example.Main"`

## Tests
Junit is used for test cases.

## Dependencies
- OpenJDK 21.0.5
- Maven 3.87
- Junit 4.13.2
- Jupiter 5.11.14
